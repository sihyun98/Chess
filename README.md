# Chess pieces
***
## Pawn
- 8개
- 앞 칸이 비면 앞으로 한 칸 직진 가능
- 시작 포지션에서는 두 칸 직진 가능
- 옆이나 뒤로는 움직일 수 없음
- 바로 앞 칸에 기물이 있으면 움직일 수 없음
- 다른 기물을 잡을 때 대각선으로 한 칸 이동 가능

## Knight
- 2개
- 옆으로 한 칸, 앞으로 두 칸 움직일 수 있음
- 옆으로 두 칸, 앞으로 한 칸 움직일 수 있음
- 모든 방향으로 이와 같이 움직일 수 있음
- 다른 기물을 뛰어넘을 수 있음

## Bishop
- 2개
- 대각선으로 원하는 만큼 움직일 수 있음
- 다른 기물을 뛰어넘을 수 없음

## Rook
- 2개
- 직선으로 원하는 만큼 움직일 수 있음
- 다른 기물을 뛰어넘을 수 없음

## Queen
- 1개
- 직선과 대각선으로 원하는 만큼 움직일 수 있음
- 다른 기물을 뛰어넘을 수 없음

## King
- 1개
- 전후좌우 대각선 어느 방향이든 한 칸 움직일 수 있음
- 다른 기물을 뛰어넘을 수 없음
- 상대에게 잡힐 수 있는 칸으로 움직일 수 없음
